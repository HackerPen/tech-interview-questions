## Contains Duplicate

Given an integer array, return true if there is any value that appears more than once in the array. Return false if every element is unique

### Examples

Example 1
```
Input: nums = [1,2,3,1]
Output: true
```

Example 2
```
Input: nums = [1,2,3,4]
Output: false
```

Example 3:
```
Input: nums = [1,1,1,3,3,4,3,2,4,2]
Output: true
```
