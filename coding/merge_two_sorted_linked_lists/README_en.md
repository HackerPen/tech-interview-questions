## Merge Two Sorted Linked Lists

Given two sorted linked lists, merge the two lists into one sorted list
Return the merged linked list


### Examples

Example 1
```
Input: list1 = [1,2,4], list2 = [1,3,4]
Output: [1,1,2,3,4,4]
```

Example 2
```
Input: list1 = [], list2 = []
Output: []
```

Example 3:

```
Input: list1 = [], list2 = [0]
Output: [0]
```
